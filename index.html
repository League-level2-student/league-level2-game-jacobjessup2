Gravity Catcher

by: Jacob Jessup

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

At first I wanted to create a game like Muse Dash but decided against it. After that I decided to create a simple game with enemies, power-ups, and power-downs.

2. What I Learned

From user testing I ended up changing the color of enemies and changed the power-ups to only require one object.

3. Architecture of My Game

The Game class holds the frame. The Panel class holds most of the code and allows everything to work. The Enemies, Power-ups, and Power-downs classes mostly work the same. The ObjectStuff class sorts between the different classes and spawns the respective classes objects. The ObjectStuff class also holds the Score and HighScore variables, and also calculates what objects are touched. The Objects class creates collision boxes around the different objects.

4. Technical Implementation

I'm proud that I was able to make a game that works fine since I've never made a game on my own in anything other than Scratch.

5. Most Challenging Part

The most challenging part of this project was figuring out how to start. I ended up using the League Invaders instructions to create the foundation of the game.

6. Future Fixes and Updates

In the future I would like to find a way to optimize it more for lower end computers. I would also in the future like to add a new gamemode to the game.
